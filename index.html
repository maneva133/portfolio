Projects

Hotel Management

Description: The Hotel Management System is a comprehensive and user-friendly full-stack application built to streamline hotel operations, including guest reservations, customer records, and service management. Developed using the Spring Boot framework on the backend and React on the frontend, this application centralizes all essential hotel functions into a single platform. It leverages modern technologies like PostgreSQL for data storage, Docker for containerization, and Kubernetes for orchestration and scalability. The entire system is integrated with a CI/CD pipeline using GitHub Actions to ensure smooth and automated deployments.

Impact: This solution empowers hotel administrators to efficiently manage day-to-day operations, reduces manual overhead, and enhances the overall guest experience. Its modular architecture and cloud-native setup make it ideal for businesses aiming to embrace digital transformation and scale seamlessly.

User Tracking Chrome Extension

Description: The User Tracking Time Chrome Extension is a personal productivity tool designed to help users monitor and manage their screen time effectively. Inspired by my own need to reduce excessive computer usage, this extension tracks website activity and visualizes time spent online using interactive charts. Built with JavaScript for the interface, Spring Boot for handling backend logic, and Chart.js for data visualization, it provides users with clear and actionable insights into their browsing habits. The extension encourages users to take regular breaks and build healthier digital routines.

Impact: This tool promotes mindful screen usage and supports better time management by making users aware of how their time is spent online. Its clean interface and informative charts turn raw data into useful feedback, making it an ideal companion for students, professionals, or anyone looking to reduce digital distractions and improve productivity.

MSE Data Scraping and Analysis

Description: The MSE (Macedonian Stock Exchange) Data project is a team project and a comprehensive data scraping, storage, and analysis system designed to provide insights into stock market trends and investor behavior. The project involved scraping data for every stockholder from the Macedonian Stock Exchange website, storing it in a database, and building an API to connect the backend with a frontend interface. The system also incorporates various financial models and oscillators to generate signals and statistics for informed decision-making.

Impact: The MSE Data project provides valuable insights into the Macedonian stock market, empowering investors with the tools they need to make informed decisions. By combining data scraping, database management, and advanced financial analysis, this project demonstrates my ability to handle complex, multi-faceted challenges and deliver practical, real-world solutions.
